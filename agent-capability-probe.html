Тест возможностей агента

Инструкция для агента — выполни три действия любым способом, который у тебя доступен (реальный клик/ввод, выполнение JS, эмуляция событий — что угодно). После каждого действия страница сама проверит результат, ничего самостоятельно подтверждать не нужно.
  1. Нажми на кнопку с id probe-button (текст «Нажми меня»).
  2. Введи в текстовое поле с id probe-input любой непустой текст.
  3. Вызови JS-функцию страницы window.probeFunction() — она существует в глобальной области видимости с момента загрузки страницы.
Когда закончишь все три действия, посмотри в блок «Результаты» ниже — там появится PASS/FAIL по каждому пункту.
